Help

Add nodes to the network using the graph search function above. Search results will be listed on entering a search term. Choose an item from the list and click "Add" to add it to the graph.

Graph display
You can drag nodes to change the graph layout, and zoom in and out to reposition the graph display.
Graph traversal
On right-click (or click and hold), a context wheel is displayed with these options:
  • add incoming/outgoing connections (when on a node)
  • undo last change
  • redo last change
  • log node
Individual selected nodes can be deleted with the delete key. Press Ctrl + Z/Y to undo/redo any changes to the graph.
Nodes and edges
Nodes labels comprise the class of node, followed by a brief descriptive label; edge labels show the property label only; see the Help pages for a key to node colours and shapes.
Node properties
An overview of a node's properties is displayed next to the node on hover and in the Graph information panel on the right when clicking on a node.
