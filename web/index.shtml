The Romantic Period Poetry Archive—or RPPA [Listen] for short—is a new, open access digital platform for global Romantic-period poetry — start your journey (almost) anywhere in the world!

Start exploring

RPPA is a rapidly prototyped and openly developed DH project, currently in public alpha. It will initially run from late 2020 to the end of 2025, when it will have reached beta stage.
